# Quick Start Guide

Get up and running with Game of Life TUI in seconds!

## Install

```bash
pip install -e .
```

## Run

```bash
game-of-life
```

## First Steps

### 1. Generate Random Pattern
Press `R` to fill the grid with random cells

### 2. Start Simulation
Press `P` to play

### 3. Adjust Speed
Press `+` to speed up, `-` to slow down

### 4. Pause and Edit
Press `P` to pause, then:
- Use **arrow keys** to move cursor
- Press **Space** to toggle cells
- Press **1-5** to spawn preset patterns

### 5. Experiment
- Press `C` to clear the grid
- Press `R` again for a new random pattern
- Try spawning multiple gliders with `1`

## Quick Reference

```
P         = Play/Pause
S         = Step one generation
R         = Random cells
C         = Clear grid
+/-       = Speed control
1-5       = Preset patterns
Ctrl+S    = Save
Q/Esc     = Quit
```
